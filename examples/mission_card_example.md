## About this card

This file is an example of the `seal_card.md` generated by `sao run`.

Each recorded mission produces:

| File | Description |
|---|---|
| `manifest.json` | Timing, command, branch, exit code, and changed file list |
| `stdout.txt` | Full command output |
| `stderr.txt` | Full command error output |
| `git_status_before.txt` | Repository status before the command ran |
| `git_status_after.txt` | Repository status after the command ran |
| `git_diff.patch` | Unified diff of all uncommitted changes |
| `seal.json` | SHA256 seal covering manifest, archive, and session directory |
| `seal.txt` | Human-readable version of the seal |
| `seal_payload.json` | Compact machine-readable mission snapshot |
| `seal_card.md` | This file — shareable Markdown mission card |
| `seal_card.html` | Standalone HTML card (browser and screenshot ready) |
| `seal_qr_payload.json` | Compact JSON for QR encoding |
| `seal_qr_payload.txt` | Same compact JSON as plain text |
| `mission_summary.md` | Human-readable session summary including stdout excerpt |

The archive `<mission_id>.zip` contains the six raw data files and is the artifact covered by `archive_sha256` in `seal.json`.
